## 🚀 Instalación y uso

Clona este repositorio:

```bash
git clone https://github.com/wen-27/Portafolio-.git
```

Entra en la carpeta del proyecto:

```bash
cd Portafolio-
```

Instala las dependencias:

```bash
npm install
```

Ejecuta el proyecto en modo desarrollo:

```bash
npm run dev
```

Luego abre en tu navegador:

```bash
http://localhost:3000
```
